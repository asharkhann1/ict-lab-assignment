# My Personal Portfolio

A clean personal portfolio built using pure html5.

## Project Sections
1. **Header**: Contains my name.
2. **About Me**: A brief introduction to my background and interests.
3. **Projects**: A list of my top three applications..
4. **Contact**: Links to my profiles and direct email.

## How to Run This Locally

Since this is a static HTML project, you don't need to install any servers or compilers. Just use live server in vscode extension or directly open it in a web browser.

1. **Clone the repository:**
   ```bash
   git clone https://github.com/asharkhann1/ict-lab-assignment.git
   ```
2. **Navigate to the folder:**
```bash
cd your-repo-name
```
3. **Open the file:**
- Simply double-click the index.html file in your file explorer.
- OR right-click the file and select "Open with" and choose your preferred web browser (Chrome, Brave, Safar etc.).
